E -> Encapsulamento: Criar um padrão de código, onde voce precisa seguir um padrao pra criar uma classe, um codigo padronizado. Protege o codigo, evita com que alguem faça besteira.
H -> Herança: Poder criar classe que herda infomações de outra classe
P -> Polimorfismo: Muitas formas! Quando um objeto pode se comportar de formas diferentes.
A -> Abstração: Ja uso, pegar informações relevantes. Pegar oq é relevante da classe pessoa: Nome, idade.
